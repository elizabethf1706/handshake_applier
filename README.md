This project basically parses jobs on handskakes. it iterates through jobs and as it goes one by one it pulls the job description and title and uses ai to see if
its relevant to what you are looking for. 
If you are not me, I suggest you change the prompt to your liking in ai.py to say waht you are looking for and what your experience is.
For setting up a set list to iterate, go to handshake, login, and then set some parameters. Once you do that (for location, job type, etc) it will have a link
that you can copy paste into this code. The only change you need to make is to the page part of the link where when it says page=1 etc you put page={page}.

I usually run it to do ALL remote jobs and internships,
then a new link for jobs within a 100 mile radius of my hometown
then internships within a 100 mile radius of my hometown.
